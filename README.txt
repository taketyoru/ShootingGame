//操作方法//
スペースで弾を打つ
方向キー→で右に移動
方向キー←で左に移動
ENTERキーで終了、ステージクリア後に押すと次のステージに進む

//クリア条件//
・木星みたいなやつを倒し,周りの敵が画面からいなくなったらクリア
・ENTERキーで次のレベルのステージへ

//敵情報//
・木星みたいなやつ(EnemyBase)
HP20　score10 レベルが上がるごとにHP1増加,移動速度1増加

・棒みたいなやつ(StraightEnemy)
HP5 score1 レベルが上がるごとにHP1増加

・ランダムに動くやつ(RandomEnemy)
HP1 score1 レベルが上がるごとにHP1増加,移動する幅が増加

・真下に速く落ちてくるやつ(DropEnemy)
HP1 score1 レベルが上がるごとにHP1増加,落ちてくる速さが1ずつ増加（めっちゃ速い）

・はぐれメタル(haguremetal)
HP4 score10050 レベルが上がってもステータス変動なし(めっちゃ強いから)
